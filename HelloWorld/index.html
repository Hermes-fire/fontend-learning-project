Au cours de cette formation nous allons avoir 2 matieres :

L'integration

Le HTML :

le HTML sert a creer le contenu du site : uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique a notre site : Les couleurs, les formes, le fond, etc.

Le Developpement

Le Javascript :

Javascript est un langage de programmation de script employe pour rendre une page web interactive, le comportement de la page et son apparence pourra varier en fonction de certains parametres.

Le PHP :

Le PHP est un langage de programmation libre, utlise pour genere des pages web dynamiques, qui sont donc modifies en fonction de certaines informations (heure, nom de l'utilisateur, formulaire rempli pae l'utilisateur etc).

MySQL :

MySQL est un systeme de gestion de bases de donnee, il permet d'enregistrer des informations rempli par l'utilisateur (exemple : formulaire de creation de compte) pour ensuite etre re-uutilise au moment approprie (ex: connexion a un site internet avec identifiant et mot de passe).
